星云大师亲身经历 揭示人的无限潜能

2017-01-09 20:50:49　来源: 网易佛学


我生性原本胆怯。年少时，对着群众说话老是舌头打结，于是，我在上台以前，将讲稿一读再读，直至滚瓜烂熟为止。平时，学习主动迎接宾客，并事先想好应对的词语。夜深，不论多么疲惫，都要自己反省一日说话的得失。


我的聪明才智比不上别人。就读佛学院时，老师的责备、同窗的讪笑，将之视为理所当然，发愿以劳动弥补己之不足。我摸黑起床，打板司钟（负责提醒作息的钟板号令）；同学自修时，我发心到河边打水供养大众；三餐前后，赶去斋堂行堂洒扫；课余之暇，前往大寮典座（在厨房担任料理大众斋饭饮食、调味的执事），在热炉沸汤、柴米油盐中穿梭不停。


丛林生活十年，我做了六年行堂（斋堂中负责照顾大众用斋的人员）、两年司水（ 准备漱洗用水的人员）、一年半的香灯（照管香烛、油灯，摆设供器、供品，清洁佛像、佛殿的人员），还兼任图书管理员。行堂是最苦的，每到冬天，我的双手浸泡在冰冻的水里洗几百双碗筷，手掌、手背的皮肤处处都龟裂了，连里面红色的肉都看得一清二楚。


为了磨炼身心，激发自己的无限潜能，我曾效法古德，以各种方式来刻苦自励。“过午不食”让我体悟到精神超脱的法喜，除去口腹贪求之欲；“刺血写经”让我感受到自己与佛陀血肉相连， 与众生心心相印；实行禁语期间，曾因多次违禁而掌掴自己，久而久之，连心中也不复闲言杂语；在拜佛礼忏期间， 我仆倒佛前，长跪哀悔往世罪业，任无明层层剥落。


一般人祈愿，莫不为自己求。我在2 0 岁以前，也不免跟大家一样， 总是祈求佛陀加持我， 让我聪明，让我进步， 让我冲破一切难关， 让我顺利学佛求道。2 0 岁以后，我从佛学院结业出来，忽然觉得每天都是为着自己向佛菩萨求这求那，会不会太自私了？此后，改为替父母师长、亲朋好友， 乃至为有缘信徒祈求，愿他们身体平安，福慧增长。


慢慢到了4 0 岁以后，有一天，我发觉这样仍是自私的贪求。于是，从4 0 岁到5 0 岁，我就为世界和平、国家富强、社会安乐、众生得度而求。过了5 0 岁，我又忽然心有所感，每天都要佛菩萨为世界、为社会、为众生成就和平、安乐，那我自己是做什么的呢？所以，5 0 岁以后，我求佛菩萨，让我来为天下众生负担业障苦难，让我来承受世间人情的辛酸冷暖，让我来体会力行佛陀的大慈大悲，让我来学习佛陀如何示教利喜。


般若讲究证悟，悟还不是证，悟是契入道理，还不能完全合二为一。证，最高的道理，如梁武帝见达摩祖师，梁武帝问：“我度生、印经，有多少功德？”“了无功德。”达摩祖师回答的，和梁武帝所了解的功德不一样。梁武帝有没有功德？当然有，世间功德，不论做多做少都有功德，只是，有相的功德，不能和很大的根器、究竟的、无为无证的、通天彻地的功德相比，达摩祖师讲的是自性功德，那是我们苦心追求的，以现在的话来说，找到无限的潜能，本领就高，武功就强，等于武侠小说常说的“打通任督二脉”。


如今， 有时听到人家说佛光山大， 我反要问：大在哪里？


道场的多寡、组织发展的规模，随缘自在吧。我当然不是7 0 岁才学佛的， 但可说到了7 0 岁才成熟，许多道理原来都在我身边，以前雾里看花，总有所求有所愿，但是，自在是不能求的，找到般若真心的本然，才一天一天，越发自在了。


经典都以“ 如是我闻” 为开头， 以“ 信受奉行”作为结束；能够信受奉行佛法，就是“行佛”。“行佛”比对佛法知识的理解或是经文的背诵，更能贴近佛陀的意旨，“行佛”，就是实践佛陀的教法。平时我们称呼学佛的人为“行者”，就是要去“修行”佛法，要如佛陀所说、所行去做，所以真正的修行人，是要“行佛”，而不只是“学佛”而已；想要有所成就，是要时时刻刻、每日地在生活中实践，而不应知道了诀窍却束诸高阁。


读《金刚经》，是为了在生活行佛、在现世实用；成就的秘诀，就在我们能否以“般若智”“金刚心”活出生命的大自在。
